# Cancer Prediction System

A Machine Learning-based web application that predicts cancer risk using patient health and lifestyle information.

## Features

* Cancer risk prediction using Random Forest Classifier
* Real-time prediction through Flask API
* User-friendly web interface
* Risk percentage calculation
* Personalized health recommendations

## Technologies Used

* Python
* Flask
* Scikit-learn
* Pandas
* NumPy
* HTML
* CSS
* JavaScript

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python app2.py
```

Open:

```text
http://127.0.0.1:5000
```

## Project Workflow

1. Data Preprocessing
2. Model Training using Random Forest
3. Model Serialization with Pickle
4. Flask Backend Integration
5. Frontend Development
6. Real-time Cancer Risk Prediction
